# Docker Integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `backend/Dockerfile` for Railway deploys and `docker-compose.yml` for one-command local development.

**Architecture:** Multistage Dockerfile builds a minimal JRE-only backend image. docker-compose.yml runs the backend via Docker and the frontend via `npm run dev`, both on their standard ports. The browser reaches the backend on `localhost:8080` directly.

**Tech Stack:** Docker, Docker Compose, eclipse-temurin:21, maven:3.9, nginx:alpine (not used for frontend — npm run dev only), Spring Boot JAR

## Global Constraints

- Java 21 (eclipse-temurin:21-jre-alpine for runtime)
- Maven 3.9 (maven:3.9-eclipse-temurin-21-alpine for build)
- Backend port: 8080
- Frontend port: 3000
- `NEXT_PUBLIC_API_URL=http://localhost:8080` in docker-compose (browser-visible, not Docker-internal)
- Working directory for all commands: `/Users/tr1-galki/test/neiro-test/v0-test`

### Task 1: backend/Dockerfile

**Files:**
- Create: `backend/Dockerfile`

**Interfaces:**
- Produces: Docker image that runs the Spring Boot backend on port 8080

- [ ] **Step 1: Create the Dockerfile**

Create `backend/Dockerfile`:

```dockerfile
FROM maven:3.9-eclipse-temurin-21-alpine AS build
WORKDIR /app
COPY pom.xml .
COPY src ./src
RUN mvn package -DskipTests

FROM eclipse-temurin:21-jre-alpine AS run
WORKDIR /app
COPY --from=build /app/target/*.jar app.jar
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
```

- [ ] **Step 2: Build the image locally to verify it works**

### Task 2: docker-compose.yml

**Files:**
- Create: `docker-compose.yml` (in `v0-test/` root)

**Interfaces:**
- Consumes: `backend/Dockerfile` from Task 1
- Produces: `docker compose up` starts both services

- [ ] **Step 1: Create docker-compose.yml**

Create `docker-compose.yml` at `/Users/tr1-galki/test/neiro-test/v0-test/docker-compose.yml`:

```yaml
services:
  backend:
    build: ./backend
    ports:
      - "8080:8080"

  frontend:
    image: node:20-alpine
    working_dir: /app
    volumes:
      - ./frontend:/app
      - /app/node_modules
    ports:
      - "3000:3000"
    environment:
      - NEXT_PUBLIC_API_URL=http://localhost:8080
    command: sh -c "npm install && npm run dev"
    depends_on:
      - backend
```

- [ ] **Step 2: Start both services**

```bash
cd /Users/tr1-galki/test/neiro-test/v0-test
docker compose up
```

Expected: backend logs show `Started ClickerApplication`, frontend logs show `Ready in Xms` on port 3000.
